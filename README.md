# Lumina Growth 💎

Lumina is a premium personal development application built with **Jetpack Compose**. It is designed to be a high-end showcase project, featuring advanced animations, premium glassmorphism aesthetics, and a clean MVVM architecture.

![Lumina Dashboard](https://raw.githubusercontent.com/placeholder/lumina_dashboard.png)

## 🌟 Key Features

### 🎨 Visual Excellence
- **Aurora Animated Background**: Dynamic, smooth color blobs moving in the background, creating a living interface.
- **Stardust Atmosphere**: Custom-drawn particle systems adding depth and texture to every screen.
- **Advanced Glassmorphism**: High-fidelity frosted glass components with fine-tuned border gradients.

### 🧘 Dashboard & Master Ring
- **Master Progress Ring**: A multi-layered, nested circular progress system that visualizes your overall daily growth across categories (Mind, Body, Knowledge, Gratitude).
- **Animated Affirmations**: Pulsing affirmation cards with inner glow effects to start your day with positivity.

### ⚡ Smart Habit Tracking
- **Staggered Entry Animations**: List items fluidly slide into view as the screen loads.
- **Celebration Confetti**: Physics-based colorful particle explosions triggered upon habit completion.
- **Visual Feedback**: Real-time progress updates with custom circular indicators.

### 📊 Mood Analytics
- **Glowing Line Chart**: A hand-drawn (Canvas) line chart visualizing emotional trends over time.
- **Interactive Logging**: Smooth transitions between different mood states and historical logs.

### 🌬️ Mindfulness Egzersizleri
- **Breathing Experience**: An immersive breathing guide with multi-layered radial gradients and organic scaling animations.

## 🛠️ Technical Stack
- **Language**: 100% Kotlin
- **UI**: Jetpack Compose (Material 3)
- **Architecture**: MVVM + Clean Architecture principles
- **Navigation**: Compose Navigation (Type-safe)
- **Dependency Injection**: Dagger Hilt
- **Asynchronous**: Kotlin Coroutines & Flow
- **Drawing**: Custom Canvas implementations for Charts, Particles, and Progress Rings.

## 🚀 Getting Started
1. Clone the repository.
2. Open with Android Studio (Ladybug or newer).
3. Run the app on an emulator or physical device (API 24+).
